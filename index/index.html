혁신적인 태양광 자동 추적 시스템

작동 원리

조도센서에서 측정된 빛의 세기를 계산하여 빛의 세기가 가장 강한 방향으로 태양광 패널이 향할 수 있도록 서브 모터를 이용해 태양광 패널을 움직입니다.

ex) 디스플레이 모듈과 네오픽셀 패널을 통해 수치를 확인할 수 있습니다.

구성 요소

  • 나노 328p 호환보드
  • 나노 I/O확장 프로토 쉴드
  • 조도센서
  • 태양광 패널
  • 서브모터
  • OLED 디스플레이 모듈
  • 네오픽셀 일자형 패널

장점 및 특징

태양열 키트는 조도센서를 이용해 일사량을 감지하고, 패널이 태양 방향을 자동 추적하도록 솔리 트리거 메커니즘을 활용해 효율을 높입니다.
구조적으로는 단축축 혹은 이축형으로 설계되어 고정형 대비 더 많은 태양광을 확보하며, 움직이는 구동부가 포함되어 자동으로 각도 조정을 실행합니다.
센서 + 제어기 + 모터/액추에이터로 구성되어 있으며, 조도센서로 빛의 강도를 측정해 패널을 움직이도록 설계되어 있습니다.
이러한 키트는 설치면적이 제한된 장소에서 출력 증가 효과가 있습니다.

수업 참여 느낌점

1803 김효겸
첨음 배우는 코딩이여서 어려울거 같고 못할거 같다는 생각이 들었지만 선생님이 설명을 잘해주셨고 친구들과 협력하여 잘 끝마칠수있었다.
1820 황혜정
태양광이나 풍력 같은 걸 배우니까 생각보다 재밌었다. 환경을 지키기 위해 태양광을 많이 설치해야한다고 느꼈다.
1805 박민석
이번 태양광 수업을 듣기 전부터 코딩을 해보고 싶다는 생각을 했었는데 수업을 통해 코딩을 정해보니 쉽게할 수 있는것 같아서 좋았다

클래스 설명

클래스
class HomeAutoLight: """현관 앞 자동 조명: 어두우면 켜고, 밝으면 끄기""" def __init__(self, light_sensor, switch): self.light_sensor = light_sensor # 주변 밝기 센서(lux) self.switch = switch # 전등 스위치(on/off) self.threshold_lux = 80 # 이 값보다 어두우면 불을 켬 def is_dark(self) -> bool: lux = self.light_sensor.read_lux() return lux < self.threshold_lux def update(self) -> None: if self.is_dark(): self.switch.turn_on() else: self.switch.turn_off() # 사용 예시 # auto = HomeAutoLight(light_sensor, light_switch) # auto.update() # 집에 들어올 때 자동으로 불을 켜거나 끕니다

코드 설명

HomeAutoLight 동작 개요
- 목적: 현관 앞 주변이 어두우면 자동으로 전등을 켜고, 밝아지면 끕니다.
- 핵심 속성: light_sensor(주변 밝기 측정), switch(전등 on/off), threshold_lux(판단 기준).
- is_dark(): 센서로 읽은 밝기(lux)가 기준값 미만인지 확인합니다.
- update(): 어두우면 turn_on(), 밝으면 turn_off()를 호출해 전등을 제어합니다.
- 사용법: 센서/스위치 객체를 주입해 update()만 주기적으로 호출하면 됩니다.
- EX) bool(불리언)은 함수의 반환값으로 참/거짓 둘 중 하나를 알려줍니다.
